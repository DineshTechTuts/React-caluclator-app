Creating Caluclator App in React JS Code

=> Creating App 
   #npx create-react-app <App_name>      (my case App_name = Caluclator)

=> Go to created directory
   #cd <App_name>                        (enter into created app directory)

=> install bootstrap 
   #npm i bootstrap                       (Installs bootstrap for app)
   
=> Start the npm server after installation
   #npm start                            (This will start the react server in localhost:3000)
